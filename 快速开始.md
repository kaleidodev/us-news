# US News AI 总结器 - 快速开始指南

## 📖 项目概述

这是一个基于 Figma 设计稿实现的 **US News Money AI 总结器** 组件库。包含三个主要组件，可直接用于生产环境。

---

## 🎯 核心组件

### 1️⃣ `AISummarizer` - 内联总结器
用于文章内嵌的紧凑式 AI 总结按钮

```tsx
import { AISummarizer } from '@/app/components';

export default function ArticleArticle() {
  return (
    <div>
      <h1>我的文章标题</h1>
      <AISummarizer articleTitle="我的文章" />
    </div>
  );
}
```

**特性：**
- 点击展开/折叠
- 完整的 AI 回复面板
- 后续问题建议
- 实时反馈系统

---

### 2️⃣ `AISummarizerLarge` - 大型面板
全屏幕专业面板，用于深度交互

```tsx
import { AISummarizerLarge } from '@/app/components';

const articles = [
  { id: '1', title: '文章1' },
  { id: '2', title: '文章2' },
];

export default function Page() {
  return <AISummarizerLarge isExpanded={true} relatedArticles={articles} />;
}
```

**特性：**
- 专业的蓝色界面
- 相关文章推荐
- 带编号的回复
- 可交互的问题建议

---

### 3️⃣ `ArticlePage` - 完整文章页面
包含导航、文章内容和 AI 总结器的完整页面

```tsx
import { ArticlePage } from '@/app/components';

export default function Page() {
  return (
    <ArticlePage 
      title="5 Best Nuclear Energy Stocks"
      author="Marc Guberti"
      date="Jan. 23, 2026"
    />
  );
}
```



## 💡 常见用法

### 在文章中使用
```tsx
// app/article/page.tsx
import { ArticlePage } from '@/components';

export default function ArticleView() {
  return <ArticlePage title="您的文章标题" />;
}
```

### 自定义相关文章
```tsx
<AISummarizerLarge 
  isExpanded={true}
  relatedArticles={[
    { id: '1', title: '比特币 ETF 指南' },
    { id: '2', title: '股票投资初学者' },
  ]}
/>
```

### 响应内容变化
```tsx
'use client';

import { useState } from 'react';
import { AISummarizer } from '@/components';

export default function DynamicArticle() {
  const [article, setArticle] = useState({ title: '...' });
  
  return <AISummarizer articleTitle={article.title} />;
}
```

## 🐛 常见问题

### Q: 依赖冲突错误？
**A:** 使用 `--legacy-peer-deps` 标志：
```bash
npm install --legacy-peer-deps
```

### Q: 端口 3000 已被占用？
**A:** Next.js 会自动使用 3001 端口，无需手动处理。

### Q: 如何修改 AI 回复内容？
**A:** 编辑组件中的硬编码文本，或从 props 传入动态内容：
```tsx
interface AISummarizerProps {
  aiResponse?: string;
}
```

### Q: 图片怎么显示？
**A:** 使用 `<img>` 标签或 Next.js Image：
```tsx
import Image from 'next/image';

<Image src={imagePath} alt="描述" width={300} height={200} />
```
